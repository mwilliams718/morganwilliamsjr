# Morgan C. Williams, Jr. — Academic Website

Static academic website hosted on GitHub Pages.

## Structure

```
morgan-williams-site/
├── index.html          # Homepage
├── research.html       # Research / papers
├── teaching.html       # Courses taught
├── cv.html             # Curriculum Vitae
├── .nojekyll           # Tells GitHub Pages to skip Jekyll processing
├── assets/
│   ├── css/style.css   # Main stylesheet
│   ├── js/main.js      # Navigation script
│   ├── img/
│   │   └── headshot.jpg    ← ADD YOUR PHOTO HERE
│   └── files/
│       └── cv.pdf          ← ADD YOUR CV PDF HERE
```

## How to Add Your Photo & CV

1. Save your headshot as `assets/img/headshot.jpg`
2. Save your CV as `assets/files/cv.pdf`

## Deploying on GitHub Pages

1. Create a new repository on GitHub (e.g., `yourusername.github.io` for a user site)
2. Upload all these files to the repository
3. Go to **Settings → Pages** and set the source to the `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io`
